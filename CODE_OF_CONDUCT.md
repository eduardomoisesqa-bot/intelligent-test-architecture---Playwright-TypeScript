# Code of Conduct

## Our Commitment

We are committed to creating a respectful, inclusive and collaborative environment for everyone interested in Quality Engineering, test architecture and automation.

## Expected Behavior

Participants are expected to:

- Communicate respectfully.
- Provide constructive feedback.
- Be open to different perspectives.
- Focus on technical arguments, not personal opinions.
- Help others learn and grow.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment or discrimination.
- Offensive comments.
- Personal attacks.
- Sharing sensitive information.
- Disrespectful communication.

## Project Culture

This project values:

- Learning
- Engineering discipline
- Technical excellence
- Collaboration
- Documentation
- Continuous improvement

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions and documentation.
